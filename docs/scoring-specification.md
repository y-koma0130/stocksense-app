# スコアリング仕様 v2

## 概要

### 目的
「割安だが中期・長期で上昇する可能性のある銘柄」を精度高く検出する

### v1からの主な変更点
1. 市場別の重み配分を導入（補正係数方式を廃止）
2. PBRの下限ペナルティ追加（バリュートラップ回避）
3. テーマタグ・マクロタグのスコアリング活用（LLM分析連携）
4. グロース市場向けロジックの最適化
5. 長期スコアにROEスコアを追加

---

## 除外ルール

### PROマーケット除外
市場名に "PRO" または "プロ" が含まれる銘柄を除外

### 罠銘柄除外（市場別）

#### プライム市場
以下のいずれかに該当で除外:
- 短期平均出来高 ≤ 30,000株/日
- 自己資本比率 < 25%
- ROE < 3%
- 営業利益が3年連続減少
- 営業CFが2年連続マイナス

#### スタンダード市場
以下のいずれかに該当で除外:
- 短期平均出来高 ≤ 7,000株/日
- 自己資本比率 < 20%
- 営業利益が2年連続減少
- 営業CFが2年連続マイナス

#### グロース市場
以下のいずれかに該当で除外:
- 短期平均出来高 ≤ 5,000株/日
- 自己資本比率 < 10%
- 営業CFが3年連続マイナス
- 売上が3年連続減少

## 市場別スコア重み配分

### 中期スコア（1〜6ヶ月投資）

| 指標 | プライム | スタンダード | グロース |
|------|----------|--------------|----------|
| PER | 24% | 26% | 15% |
| PBR | 18% | 20% | 5% |
| RSI（14週） | 16% | 16% | 18% |
| 価格位置（26週） | 12% | 12% | 15% |
| RSIモメンタム | 18% | 16% | 17% |
| 出来高急増 | 12% | 10% | 10% |
| EPS成長率 | - | - | 12% |
| タグスコア | - | - | 8% |
| **合計** | **100%** | **100%** | **100%** |

#### 中期の市場別設計意図

**プライム**
- バリュエーション指標（PER/PBR）を重視
- 大型株は適正価格への収束が期待できる

**スタンダード**
- プライムに近い構成だが、PER/PBRをやや重視
- 割安放置されやすい市場特性を考慮

**グロース**
- PER/PBR重みを大幅削減（低バリュエーション＝成長鈍化シグナルの可能性）
- EPS成長率を追加（成長継続の確認）
- モメンタム系指標を重視（トレンドフォロー）
- タグスコアで旬のテーマを加味

## 指標別スコア閾値

### PERスコア

| 条件（業種平均比） | スコア |
|-------------------|--------|
| 70%以下 | 100点 |
| 70%〜100% | 100→50点（線形補間） |
| 100%〜150% | 50→0点（線形補間） |
| 150%超 | 0点 |

**除外条件**: PER ≤ 0、業種平均PER ≤ 0 の場合は0点

---

### PBRスコア

#### 基本スコア（業種平均比）

| 条件（業種平均比） | スコア |
|-------------------|--------|
| 40%未満 | 60点（低すぎ警告） |
| 40%〜70% | 100点 |
| 70%〜100% | 100→50点（線形補間） |
| 100%〜150% | 50→0点（線形補間） |
| 150%超 | 0点 |

#### 絶対値ペナルティ（基本スコアに乗算）

| 条件 | ペナルティ |
|------|-----------|
| PBR < 0.3倍 | スコア × 0.7（30%減点） |
| PBR < 0.5倍 かつ ROE < 5% | スコア × 0.8（20%減点） |

**除外条件**: PBR ≤ 0、業種平均PBR ≤ 0 の場合は0点

**設計意図**:
- PBR 0.3倍未満は市場が「将来性なし」と評価している可能性
- PBR 0.5倍未満 かつ 低ROEは典型的なバリュートラップ

### RSIスコア

| 条件 | スコア |
|------|--------|
| 30以下 | 100点（売られすぎ） |
| 30〜50 | 100→50点（線形補間） |
| 50〜70 | 50→0点（線形補間） |
| 70超 | 0点（買われすぎ） |

**期間**:
- 中期: 14週RSI
- 長期: 52週RSI

---

### 価格位置スコア

価格位置 = (現在価格 - 期間安値) / (期間高値 - 期間安値) × 100

| 条件 | スコア |
|------|--------|
| 20%以下 | 100点（底値圏） |
| 20%〜40% | 100→50点（線形補間） |
| 40%〜100% | 50→0点（線形補間） |

**期間**:
- 中期: 26週
- 長期: 52週

**除外条件**: 現在価格・高値・安値がnull、高値=安値、価格がレンジ外の場合は0点

---

### RSIモメンタムスコア（中期のみ）

モメンタム = RSI(2週) - RSI(14週)

| 条件 | スコア |
|------|--------|
| -30以下 | 0点（強い下降モメンタム） |
| 0 | 50点（モメンタムなし） |
| +30以上 | 100点（強い上昇モメンタム） |

中間値は線形補間

**除外条件**: RSI取得不可の場合は50点（中央値）

---

### 出来高急増スコア（中期のみ）

出来高比率 = 短期平均出来高(1週) / 長期平均出来高(5週)

| 条件 | スコア |
|------|--------|
| 0.5以下 | 0点（大幅減少） |
| 1.0 | 50点（横ばい） |
| 2.0以上 | 100点（大幅増加） |

中間値は線形補間

**除外条件**: 出来高取得不可の場合は50点（中央値）

---

### EPS成長率スコア

成長率 = ((最新EPS / 3年前EPS)^(1/3) - 1) × 100（3年CAGR）

| 条件 | スコア |
|------|--------|
| 0%未満 | 0点（マイナス成長） |
| 0%〜10% | 0→50点（線形補間） |
| 10%〜20% | 50→100点（線形補間） |
| 20%超 | 100点（高成長） |

**除外条件**: EPS null、3年前EPS ≤ 0、最新EPS < 0 の場合は50点（中央値）

**使用市場・期間**:
- 長期: プライム・スタンダード・グロース全て
- 中期: グロースのみ

---

### ROEスコア（長期のみ）

| 条件 | スコア |
|------|--------|
| 5%未満 | 0点 |
| 5%〜8% | 0→50点（線形補間） |
| 8%〜15% | 50→100点（線形補間） |
| 15%超 | 100点 |

**除外条件**: ROE取得不可の場合は50点（中央値）

**設計意図**: 長期保有では収益性の持続が重要

### タグスコア

LLMマーケット分析ジョブが出力する有望・不利タグを使用

#### 入力データ形式

マーケット分析ジョブが以下を出力:

| フィールド | 説明 | 例 |
|-----------|------|-----|
| favorableThemeTags | 有望テーマタグ | ["ai", "semiconductor", "defense"] |
| unfavorableThemeTags | 不利テーマタグ | ["real_estate", "china_related"] |
| favorableMacroTags | 有望マクロタグ | ["export", "yen_weak"] |
| unfavorableMacroTags | 不利マクロタグ | ["import", "domestic_demand"] |

#### スコア計算

**一致数から加点/減点を計算**

| 一致数 | 加点/減点 |
|--------|----------|
| 0個 | 0点 |
| 1個 | 15点 |
| 2個 | 30点 |
| 3個以上 | 50点 |

**テーマタグスコア**

ベース: 50点
- 有望テーマ一致: +15/30/50点（1/2/3個以上）
- 不利テーマ一致: -15/30/50点（1/2/3個以上）
- 相殺方式: 有望と不利の加点/減点を相殺

例:
- 有望2個(+30) + 不利1個(-15) = 50 + 30 - 15 = 65点
- 有望1個(+15) + 不利1個(-15) = 50 + 15 - 15 = 50点（中立）

**マクロタグスコア**

テーマタグスコアと同様のロジック

**タグスコア合成**

- 中期（グロースのみ）: テーマスコアのみ使用
- 長期（全市場）: (テーマスコア × 0.6) + (マクロスコア × 0.4)

**スコア範囲**: 0〜100点（clamp処理）

**設計意図**:
- 複数タグに該当する銘柄をより適切に評価
- 有望と不利の両方に該当する場合は相殺で中立に近づく
- 中期は短期的な市場テーマの影響が大きい
- 長期はマクロ環境（景気サイクル、為替）の影響も考慮

## 総合スコア計算

### 計算式

総合スコア = Σ(各指標スコア × 重み) / 100

スコアは0.0〜1.0の範囲

### スコアレンジの解釈

| スコア範囲 | 評価 |
|-----------|------|
| 0.8〜1.0 | 最高（非常に割安で反転期待が高い） |
| 0.6〜0.8 | 高（割安で反転の可能性あり） |
| 0.4〜0.6 | 中（やや割安） |
| 0.0〜0.4 | 低（割安感なし or 上昇トレンド中） |
