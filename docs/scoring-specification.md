# スコアリング仕様

## 取得指標

| 指標 | 用途 | 使用期間 |
|------|------|----------|
| PER | 割安性評価（業種平均比） | 中期・長期 |
| PBR | 割安性評価（業種平均比） | 中期・長期 |
| RSI | 売られすぎ評価 | 中期: 14週、長期: 52週 |
| 価格位置 | 底値圏評価 | 中期: 26週、長期: 52週 |
| RSIモメンタム | 反発初動検出（短期RSI - 長期RSI） | 中期のみ |
| 出来高比率 | 注目度上昇検出（短期/長期出来高） | 中期のみ |
| EPS成長率 | 利益成長性評価（3年CAGR） | 長期のみ |
| 自己資本比率 | 財務安定性判定 | 罠株除外 |
| ROE | 収益性判定 | 罠株除外 |
| 営業利益減少年数 | 利益トレンド判定 | 罠株除外 |
| 営業CF負の年数 | キャッシュフロー健全性 | 罠株除外 |
| 売上減少年数 | 成長性判定 | 罠株除外（グロースのみ） |
| 短期平均出来高 | 流動性チェック | 罠株除外 |

---

## 除外ルール

### PROマーケット除外
市場名に "PRO" または "プロ" が含まれる銘柄を除外

### 罠銘柄除外（市場別）

#### プライム市場
- 短期平均出来高 ≤ 30,000株/日
- 自己資本比率 < 25%
- ROE < 3%
- 営業利益が3年連続減少
- 営業CF が2年連続マイナス

#### スタンダード市場
- 短期平均出来高 ≤ 7,000株/日
- 自己資本比率 < 20%
- 営業利益が2年連続減少
- 営業CF が2年連続マイナス

#### グロース市場
- 短期平均出来高 ≤ 5,000株/日
- 自己資本比率 < 10%
- 営業CF が3年連続マイナス
- 売上が3年連続減少

## 市場別補正係数

| 市場 | PER補正 | PBR補正 | 価格位置補正 |
|------|---------|---------|-------------|
| プライム | 1.0 | 1.0 | 1.0 |
| スタンダード | 1.05 | 1.0 | 1.05 |
| グロース | 1.2 | 0.8 | 0.95 |
| その他 | 1.0 | 1.0 | 1.0 |

**補正の意図**:
- グロース市場: PERを重視（成長期待）、PBRを軽視（資産ベース評価は不向き）
- スタンダード市場: 割安株が多い市場特性を考慮

## 中期スコアの指標と閾値

### PERスコア
```
ratio = 個別PER / 業種平均PER × 100

ratio ≤ 70%   → 100点
ratio ≤ 100%  → 100〜50点（線形補間）
ratio ≤ 150%  → 50〜0点（線形補間）
ratio > 150%  → 0点
```
除外: PER ≤ 0、業種平均PER ≤ 0

### PBRスコア
```
ratio = 個別PBR / 業種平均PBR × 100

ratio ≤ 70%   → 100点
ratio ≤ 100%  → 100〜50点（線形補間）
ratio ≤ 150%  → 50〜0点（線形補間）
ratio > 150%  → 0点
```
除外: PBR ≤ 0、業種平均PBR ≤ 0

### RSIスコア（14週）
```
RSI ≤ 30  → 100点
RSI ≤ 50  → 100〜50点（線形補間）
RSI ≤ 70  → 50〜0点（線形補間）
RSI > 70  → 0点
```

### 価格位置スコア（26週）
```
価格位置 = (現在価格 - 期間安値) / (期間高値 - 期間安値) × 100

価格位置 ≤ 20%   → 100点
価格位置 ≤ 40%   → 100〜50点（線形補間）
価格位置 ≤ 100%  → 50〜0点（線形補間）
```

### RSIモメンタムスコア
```
momentum = RSI(2週) - RSI(14週)

momentum ≤ -30  → 0点
momentum = 0    → 50点
momentum ≥ +30  → 100点
（線形補間）
```

### 出来高急増スコア
```
ratio = 短期平均出来高(1週) / 長期平均出来高(5週)

ratio ≤ 0.5  → 0点
ratio = 1.0  → 50点
ratio ≥ 2.0  → 100点
（線形補間）
```

## 長期スコアの指標と閾値

### PERスコア
中期と同じ計算ロジック

### PBRスコア
中期と同じ計算ロジック

### RSIスコア（52週）
```
RSI ≤ 30  → 100点
RSI ≤ 50  → 100〜50点（線形補間）
RSI ≤ 70  → 50〜0点（線形補間）
RSI > 70  → 0点
```

### 価格位置スコア（52週）
中期と同じ計算ロジック（期間が52週）

### EPS成長率スコア
```
growthRate = ((最新EPS / 3年前EPS)^(1/3) - 1) × 100

growthRate < 0%   → 0点
growthRate ≤ 10%  → 0〜50点（線形補間）
growthRate ≤ 20%  → 50〜100点（線形補間）
growthRate > 20%  → 100点
```
除外: EPS null、3年前EPS ≤ 0、最新EPS < 0 の場合は50点
